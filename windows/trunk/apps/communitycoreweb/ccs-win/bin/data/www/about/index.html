About CCW

Thanks for downloading!

This is an early release of a cross-platform local web development stack, based on Mongoose web server and initially including PHP, MySQL and a Web GUI which may be used with locally networked apps.

Currently CCW implements a pseudo-plugin architecture. With time the plan is to support dynamic run-time linked shared object plugins, allowing entirely an entirely customised CCW app without having to recompile anything.
